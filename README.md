# Campus Courses Records Manager (CCRM)

A Java-based console application for managing campus academic records. It simplifies student, course, enrollment, and grade management using a menu-driven interface and CSV-based storage.

## Features
- *Student Management:* Add, list, update, and search students.
- *Course Management:* Add, list, and search courses.
- *Enrollment & Grading:* Enroll students, record marks, calculate GPA, and generate reports.
- *CSV Import/Export:* Load and save student/course data.
- *Backup:* Automatic data backups for safety.
- *Reports & Analytics:* View student/course statistics and top performers.

## Technologies
- Java SE (Core Java)
- CSV-based storage (no external database required)
